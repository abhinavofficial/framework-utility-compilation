## Snapshot

| Operation    | Insert/Add   | Search/Contains         | Delete/Remove |
|--------------|--------------|-------------------------|---------------|
| Array        | O(1)         | O(n)                    | O(n)          |
| Sorted Array | O(n)         | O(logn)                 | O(n)          |
| BST          | O(H) or O(n) | O(H) or O(n) or O(Logn) | O(h) or O(n)  |
