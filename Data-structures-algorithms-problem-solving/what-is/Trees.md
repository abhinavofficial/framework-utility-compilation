## Binary Tree Traversal
* Preorder Traversal: Time Complexity: O(n)
  * Root
  * Left Subtree
  * Right Subtree
* Inorder Traversal: Time Complexity: O(n)
  * Left Subtree
  * Root
  * Right Subtree
* Postorder Traversal: Time Complexity: O(n)
  * Left Subtree
  * Right Subtree
  * Root
* Level Traversal: Time Complexity: O(n)
  * Level 1 -> Level n
